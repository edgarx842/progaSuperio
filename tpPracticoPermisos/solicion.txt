ejercicio 2
Para crear el script, se utilizó un editor de texto y se guardó como script.sh. 

Se asignaron los permisos necesarios utilizando el comando chmod 711 script.sh, 

donde:

- El propietario tiene permisos de lectura, escritura y ejecución (7).

- El grupo tiene permisos de ejecución (1).

- Otros usuarios tienen solo permisos de ejecución (1).
